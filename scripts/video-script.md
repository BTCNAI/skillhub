## 拍摄建议

### 设备
- 屏幕录制：OBS Studio (1080p 60fps)
- 麦克风：Blue Yeti 或类似
- 后期：Final Cut Pro / DaVinci Resolve

### 风格
- 快节奏剪辑
- 代码高亮用 Monokai 主题
- 背景音乐：轻快电子乐
- 转场：简洁的 fade/cut

### B-roll
- 终端操作录屏
- GitHub 页面
- 代码编辑器
- 简单的动画效果

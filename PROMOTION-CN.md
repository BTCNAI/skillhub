## 知乎

### 问题：有哪些好用的 AI 工具推荐？

**回答：**

分享一个我最近做的开源项目——SkillHub。

**问题背景：**

我在使用 OpenClaw（一个 AI Agent 框架）时，发现技能管理很麻烦：
- 技能分散在各个仓库，难以发现
- 安装需要手动复制粘贴
- 没有版本管理

**解决方案：**

做了 SkillHub，一个技能包管理器。

**使用方式：**

```bash
# 搜索技能
skillhub search weather

# 安装技能
skillhub install weather

# 查看热门
skillhub list --popular
```

**功能：**
1. 技能发现和搜索
2. 一键安装/更新/卸载
3. 热门/趋势技能排行榜
4. 技能创作模板

**技术栈：** Node.js + Commander.js

**GitHub：** https://github.com/BTCNAI/skillhub

**目标：** 1000 stars，求支持！⭐

欢迎提 issue 和 PR！
